## まず構造を押さえる ─ 「基盤モデル3社 × IDE/アシスタント2社」の二層

今の AI プロダクト市場は、**基盤モデルを作る3社**(Anthropic / OpenAI / Google)と、**それを取り込んで開発者体験を作る2社**(Cursor / GitHub Copilot)の二層構造になっています。Cursor も Copilot も、中で動くモデルは Claude / GPT / Gemini から選べるので、基盤モデルの勝者と IDE の勝者は別々に決まります。言い換えると、「Cursor にするか Claude にするか」ではなく「Cursor 上でどのモデルを使うか」という組み合わせの問題です。

この1ヶ月の動きを一言でいうと、**5社ぜんぶがエージェントに全振りした月**でした。Anthropic は Claude Opus 4.7 を 2026-04-16 に GA し、Claude Code を 10 日で 10 バージョン以上更新。OpenAI は Agents SDK 次世代版を GA にしつつ SI 7社(Accenture / Capgemini / CGI / Cognizant / Infosys / PwC / TCS)との同時提携を発表。Google は Deep Research Agent API を正式公開。Cursor 3.0 は Agents Window を前面に据え、Copilot は VS Code Autopilot の public preview を投入。自律的に長時間タスクを回す仕組みが、どの社でも「有料プランの目玉」になりました。

押さえておくべき論点は3つ。

- **エージェント化の集中投下** — 5社すべてが 2026-04 に「AI に長時間仕事を任せる」ための大型アップデートを投入した
- **運用信頼性が次の勝負どころ** — Gemini 3.1 Pro のレイテンシスパイク、OpenAI の 3〜5 日周期の短時間劣化、Antigravity の不安定さ評など、「壊れずに走り切るか」への関心が鮮明に
- **二層構造の定着** — IDE 層と基盤モデル層の勝者は独立に決まる前提で、ツール選定も「組み合わせ」で考える段階に入った

## 基盤モデル3社 ─ 戦略の分岐が鮮明に

同じ「フロンティア級 LLM」と括るのが難しくなるほど、3社は別々の方向に振り切り始めています。

### Claude ─ コーディングと長時間エージェントで一点突破

Anthropic の 2026-04 は怒涛の更新でした。Claude Opus 4.7 を 2026-04-16 に GA し、**$5/$25 per M tokens の価格据え置きでコーディング +13%、Rakuten-SWE-Bench の production task で 3 倍、BigLaw Bench 90.9%** を叩き出しています。トークナイザ刷新で最大 35% 多くのトークンを扱え、vision が 2,576px まで拡張、思考強度を上げる `xhigh` effort も追加。同時期に Claude Code は v2.1.107 → v2.1.117 まで連続リリースされ、Managed Agents($0.08/session-hour)、ant CLI、Claude Design、Advisor tool、ネイティブバイナリ化、1時間プロンプトキャッシュ TTL を畳み掛けました。

コミュニティ評価もこれを追認していて、Pragmatic Engineer のシニア調査で **「最も愛されるコーディング支援ツール」46%** を獲得、JetBrains CSAT 91% / NPS 54、2026 Developer Survey の複雑タスク選好率 44% で首位。Zenn / note でも「PR の 83% を AI レビューだけでマージ」「40 ページ資料を 3 時間で作成」といった業務導入事例が継続的に共有されています。

## コーディングツール2強 ─ 尖る Cursor と、守りの Copilot

Cursor と GitHub Copilot は同じ IDE/アシスタント層ですが、**向かう方向が明確に分岐**しました。Cursor は「エージェント中心ワークスペース」への先鋭化、Copilot は「GitHub プラットフォーム統合と規制業界対応」で底堅く攻める構図です。どちらが優れているかというより、チームがどちらの世界観に乗るかの選択になっています。

### Cursor ─ Composer 2 と Agents Window で書く体験を再定義

Cursor 3.0(2026-04-02)で Agents Window が中心に据えられ、**ローカル・worktree・クラウド・リモート SSH の並列実行**が可能に。Design Mode、Agent Tabs、`/worktree` + `/best-of-n` など計 27 の変更が一度に降ってきました。自社フロンティアの **Composer 2 は CursorBench 61.3(1.5 比 +37%)、Terminal-Bench 2.0 で 61.7、SWE-bench Multilingual 73.7** を記録しつつ、$0.50/$2.50 per M tokens という安さで提供されています。

Bugbot は独立課金 $40/seat に育ち、44,000+ 学習ルールと 110,000+ リポジトリを背景に解決率 78%。2026-03-25 に Self-hosted Cloud Agents が GA、2026-03-04 に JetBrains IDE 対応、2026-04-21 に SpaceX とのモデル学習協業を発表。**ARR 20 億ドル、DAU 1M、エンタープライズ売上は 2025 年に 100 倍成長**と公開されています。

**主要機能・プラン**

| 項目 | 内容 |
|---|---|
| 代表機能 | Composer 2、Agents Window、Design Mode、Tab Fusion、Bugbot、Self-hosted Cloud Agents |
| 採用モデル | Claude / GPT / Gemini のフロンティアに加え自社 Composer 2 と Tab Fusion |
| 主要プラン | Pro $20、Pro+ $60、Ultra $200、Teams $40/seat、Bugbot $40/seat、Enterprise(Self-hosted) |

**強み**:対話型 IDE としての「書く速度」を引き上げる UI が洗練されていて、VS Code 互換で移行ハードルが低いのも継続的に評価されています。日本企業(カカクコム 500 人規模、YOUTRUST、Ubie、ココナラ、クラウドビルダーズ等)の導入事例も厚みを増しています。

**弱み**:使用量課金の透明性に対する不満が根強く、G2 / Product Hunt / Zenn で「料金が割高」「トークン消費の可視性が低い」「想定外課金」の声が継続しています。2025-06 の料金改定炎上の余波もあり、**プラン変更時の告知品質が契約継続率の鍵**になってきています。

### GitHub Copilot ─ プラットフォーム統合で標準解へ

Copilot は GitHub エコシステム全体を巻き込む「標準ツール」の位置を固めつつあります。2026-04-13 には **Data Residency(US + EU)と FedRAMP 準拠が同日発表**され、同時に Cloud agent の merge conflict を 3 クリックで解決する機能、CLI セッションを Web/Mobile からリモート制御する preview を公開。2026-03-05 に Copilot Code Review のエージェンティックアーキテクチャが GA、2026-02 には **GPT-5.3-Codex の LTS(Business / Enterprise 向けに 2027-02-04 まで保証)** を開始しています。

VS Code Autopilot の public preview、Agent Mode の JetBrains 対応、Custom Agents(`.agent.md`)、Copilot SDK public preview、Copilot Extensions、第三者エージェント委任、GitHub Spark(Vite / React 19+ / TypeScript / Tailwind v4+ / shadcn/ui)といった拡張も揃いました。一方で 2026-04-20 に Pro / Pro+ / Student の新規サインアップが一時停止され、2026-03 末の PR tips 事件(1.5M の PR に広告的な助言が挿入され GitHub が撤回)もありました。

**強み**:IDE 網羅度が突出していて、VS Code・Visual Studio・JetBrains・Vim・Neovim・Xcode・Eclipse・Zed・Azure Data Studio・SSMS まで対応。G2 4.5/5.0(227 件、5 星 74% / 4 星 22%)と Gartner 肯定評価に支えられ、**Pro $10 は Cursor Pro $20 の半額**でコスパ面の評価が安定しています。Issue・PR・Actions・Runner・Mobile・Firewall まで含む GitHub プラットフォーム統合は他社が簡単に再現できない強みです。

**弱み**:個人プラン新規受付の一時停止や Opus 4.6 Fast の退役など、経営判断の変動が短期的な不安として残っています。Web agent の 90 秒スピンアップ、2026-02 の RoguePilot 脆弱性(Codespaces で GITHUB_TOKEN 漏洩)、2026-03 末の PR tips 事件など、**信頼性・セキュリティ面の過去事象がユーザーの記憶に残っている**のも事実です。

## 立場別の判断ガイド

**スタートアップ / 個人開発で「書く速度」を最大化したい**

Cursor Pro($20)+ Claude Pro($17〜20)の 2 本立てが機能します。Cursor の Tab / Composer で対話的な体験を持ち上げつつ、Claude の Skills / Sub-agents で「一晩回しておく」ワークロードを任せる構図です。Pro+ $60 や Ultra $200 に上げるのは、自分のトークン消費実績が見えてからで遅くありません。

**中堅テック企業で GitHub 中心の開発フローを維持したい**

Copilot Business($19/seat)を主軸に、複雑タスクや自律リファクタは Claude Team Standard($25/seat、Claude Code 込み)に委任する構図が現実的です。Issue・PR 起点のワークフローを崩さず、長時間タスクだけ Claude Code に寄せられます。

**規制業界(金融・医療・政府)で腰を据えて導入したい**

Copilot Enterprise + Claude Enterprise が順当です。FedRAMP、Data Residency(US + EU)、HIPAA-ready、IP インデムニティ、GPT-5.3-Codex LTS 保証(2027-02-04 まで)を揃えつつ、Bedrock / Vertex 経由の Claude で IDE 外の長時間タスクに備えます。監査・調達部門からの質問を想定すると、両方押さえておく方が稟議を通しやすいです。

**全社導入でエンタープライズを攻めたい**

Claude Enterprise(Partner Network $100M の支援あり)+ OpenAI Enterprise(SI 7社の支援あり)を軸に、用途別に使い分ける構図が見えてきました。Outlook / Microsoft 365 / Cowork 統合や業種特化エージェント開発は OpenAI、コーディング・クリエイティブ・報告書作成は Claude、という住み分けが現場の実感に合います。

**研究・データ分析でリサーチ精度が効く**

Google AI Ultra($249.99)+ ChatGPT Pro $100 の併用が強力です。Deep Think、NotebookLM(EPUB 生成 / Cinematic Video Overviews)、Deep Research Agent API に加え、GPT-Rosalind(生命科学、trusted access)、Atlas Browser Memories が揃います。日本語での資料要約は NotebookLM + Deep Research Agent API が体感品質で抜けています。

**クリエイティブ(画像・動画)を攻めたい**

Google AI(Veo 3.1 / Nano Banana 2 / Lyria 3 Pro)+ ChatGPT(gpt-image-2)の併用がほぼ定石。動画は Veo 3.1 の 4K / 9:16 がほぼ一択、画像は gpt-image-2 と Nano Banana 2 がテイスト差で拮抗しています。
