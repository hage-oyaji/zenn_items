## エグゼクティブサマリー

1. **3社同点首位 20 点**:Claude / OpenAI / Google AI が 10 軸の順位点(1位=3/2位=2/3位=1)合計で **20 点同点** に収束。前回(OpenAI 23 / Google 20 / Claude 17)から Claude が **+3 点で2ランクアップ**、OpenAI が **-3 点で首位陥落(同点首位)**。
2. 変動要因は3つ:**Opus 4.7 GA による Claude の技術力1位昇格**、**ChatGPT 市場シェア低下(60%→45% 未満)+ Sora Web/App 2026-04-26 終了予告**、**Google の Flash TTS / Veo 4 / Embedding 2 / Robotics-ER 1.6 集中投入**。
3. 競争軸は「モデル性能」から「**エージェント実行基盤 + 用途特化の深さ**」へ移行。意思決定は「用途と業種でマルチモデルを使い分ける」が合理解。

## 業種・用途別の意思決定ガイド

- **SaaS / スタートアップの自律コーディング**:**Claude Code を第一選択**。Opus 4.7 のトークンインフレ(xhigh で 2.1 倍消費)に注意
- **macOS 中心の自律エージェント(ブラウザ + Figma + Xcode + Slack 横断)**:**OpenAI Codex「for (almost) everything」**。EU/UK/Switzerland は Computer Use 未提供
- **規制産業(金融 / 医療 / 法務)**:**OpenAI ChatGPT Enterprise**(Compliance API / EKM / UK Residency / ZDR)。生命科学は **GPT-Rosalind**
- **Google Workspace を基盤にする組織**:**Google AI(Gemini Enterprise + Workspace 同梱)**
- **研究・教育・文献調査**:**Google NotebookLM**(EPUB 対応 / custom chat goals 10,000 字 / Gemini 3.1 Pro 統合)
- **マーケティング動画・SNS 縦型**:**Google Veo 4(Ultra)/ Veo 3.1(無料・1日12本)**。Sora は 2026-04-26 以降の新規採用非推奨
- **音声ナレーション・多言語音声**:**Google Gemini 3.1 Flash TTS**(音声対話は OpenAI Realtime API が第一選択継続)
- **大量バッチ・コスト最小化**:**Google Gemini 3.1 Flash-Lite + Batch 50% + Context Caching**。Embedding 2($0.20/1M、Batch $0.10/1M)で RAG も最安帯
- **本番常駐エージェント基盤**:**Claude Managed Agents($0.08/session-hour)** と **OpenAI Agents SDK 次世代版(追加料金なし)** を候補比較
