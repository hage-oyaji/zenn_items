## いま、AIの勢力図はこうなっている

基盤モデルを作る三社と、それを使い倒す IDE/アシスタント二社という二層構造が、この1ヶ月でかなり鮮明になった。OpenAI は ChatGPT の週次9億アクティブを土台に「ひとつのサブスクで全部任せられる」ゼネラリスト路線をさらに尖らせ、Codex 週次4百万開発者と SI 七社(Accenture / Capgemini / CGI / Cognizant / Infosys / PwC / TCS)の同時契約でエンタープライズ導線を一気に太くした。Anthropic は Claude Opus 4.7 を 2026-04-16 に GA し、Claude Code の週次更新ラッシュで「複雑な仕事はエージェントに任せる」体験を完成させつつある。Google はジェネラリスト路線を捨てて Flash 系の用途別モデル分業に振り切り、Veo 3.1 の 4K 動画や Nano Banana 2 の画像、Flash TTS の 70 言語音声など、モダリティの幅で他を引き離している。

IDE 側では Cursor が 3.0 リリースで「エージェント中心ワークスペース」を前面に押し出し、GitHub Copilot が FedRAMP 準拠と Data Residency で規制業界に踏み込んだ。両者とも Claude Opus 4.7 や GPT-5.4、Gemini 3.1 Pro を取り込める作りなので、基盤モデル層の改善がそのまま IDE 体験の向上に反映される構造になっている。言い換えれば、基盤モデル市場とコーディングツール市場の勝者は独立に決まる。

この俯瞰を支える論点を三つだけ抽出すると:

- **エージェント化の集中投下** — Opus 4.7 GA、Agents SDK 次世代版 GA、Deep Research Agent API、Cursor 3.0、Copilot VS Code Autopilot。五社すべてが 2026-04 にエージェント関連の目玉を並べた。
- **運用信頼性への関心シフト** — Gemini 3.1 Pro のレイテンシスパイク、Antigravity のレート制限、OpenAI の 3〜5 日周期の短時間劣化など、長時間走るエージェントに対する「壊れにくさ」の要求が顕在化している。
- **二層構造の定着** — Cursor も Copilot も Anthropic/OpenAI/Google の三社モデルをユーザーが選ぶ構造が当たり前になり、IDE の勝者と基盤モデルの勝者は別物として語られるようになった。

## 基盤モデル三社のポートレート

ChatGPT・Claude・Gemini の三つは、同じ「生成 AI」と括るのが難しいほど方向性が分化している。ゼネラリストで全部取りにいく OpenAI、特化と倫理的ポジションで支持を集める Anthropic、モダリティと価格で裾野を広げる Google、という三者三様の像が、この1ヶ月の動きで一段と明確になった。

### Claude ─ コーディングと長時間エージェントで「最も愛される」ポジションを固める

Anthropic の 2026-04 は怒涛だった。フラッグシップの Claude Opus 4.7 を 2026-04-16 に一般提供開始し、同じ $5/$25 の価格のままコーディングを 13% 押し上げ、Rakuten-SWE-Bench で production task 3 倍、BigLaw Bench 90.9% を叩き出した。トークナイザーが刷新されて最大 35% 多くのトークンを扱え、vision が 2,576px と三倍超になり、思考強度を上げる `xhigh` エフォートも追加されている。そこに Claude Code が v2.1.107 から v2.1.117 まで 10 バージョン以上を 10 日余りで連続投入し、Managed Agents($0.08/session-hour)、ant CLI、Claude Design、Advisor tool を畳み掛けた。コミュニティ評価もこれを追認していて、Pragmatic Engineer のシニア調査で「最も愛されるコーディング支援ツール」として 46% を獲得し、JetBrains CSAT 91% / NPS 54、2026 Developer Survey の複雑タスク選好率 44% で首位に立っている。

**強み**: コーディングとエージェントに一点突破で集中投資した結果、職業エンジニア層の実感評価で他を抜くポジションを固めた。「Claude is a space to think」と宣言して広告を載せない姿勢は、クリエイターや研究者にも響いていて、日本語圏の Zenn や note でも「83% の PR を AI レビューだけでマージした」「40 ページ資料を 3 時間で作成した」といった具体的な業務導入事例が連続して共有されている。

**弱み**: 動画・音声生成のようなモダリティ幅ではかなり Google に遅れており、一般消費者向けの知名度でも ChatGPT に及ばない。エンタープライズ売上でも OpenAI の SI 七社契約のような大型の囲い込みは未達で、大企業の全社展開では「第二候補」になりがち。

### OpenAI ─ 単一サブスクで全部任せる、エンタープライズ導線も一気に太く

OpenAI は相変わらずゼネラリストとして最も広い面を取りに来ている。2026-04-09 に Plus $20 と Pro $200 の中間を埋める Pro $100 ティアを新設し、2026-04-02 には Business 年額を $25 から $20 に下げ、2026-04-21 には gpt-image-2 の発表と同日に SI 七社契約 + Codex Labs(OpenAI 社員常駐型のパートナー)発表という派手な一日を演出した。ChatGPT 週次 9 億アクティブユーザー、Codex 週次 4M 開発者、Agents SDK 次世代版 GA(Blaxel / Cloudflare / Daytona / E2B / Modal / Runloop / Vercel のネイティブサンドボックス連携)、Codex の Computer Use / In-app Browser / Memory / 90+ プラグイン / Thread Automations 追加。一方で動画生成の Sora は 2026-04-26 に Web/App、2026-09-24 に API が終了予定で、コア(推論 + コード + Computer Use)にリソースを寄せる判断を明確にした。

### Google AI ─ モダリティ分業とコスパで裾野を広げる

## コーディングツール二強のポートレート

Cursor と GitHub Copilot は、どちらも Anthropic・OpenAI・Google の基盤モデルを取り込む IDE/アシスタント層で、同じ材料を使ってもプロダクトとしての体験はかなり違う。Cursor は「エージェント中心ワークスペース」への振り切りが鮮明、Copilot は「GitHub プラットフォーム統合と規制業界対応」で底堅い。どちらが上かという問いよりも、どちらの世界観に乗るかの選択になりつつある。

### Cursor ─ Composer と Agents Window で「書く体験」を再定義

Cursor 3.0 が 2026-04-02 にリリースされ、Agents Window でローカル・worktree・クラウド・リモート SSH を並列で走らせられる UI、Design Mode、Agent Tabs、`/worktree` + `/best-of-n` など計 27 の変更が一度に降ってきた。自社フロンティアモデル第 2 世代の Composer 2 は CursorBench 61.3(1.5 比 +37%)、Terminal-Bench 2.0 で 61.7、SWE-bench Multilingual で 73.7 を記録し、$0.50/$2.50 per M tokens という安さで提供されている。Bugbot は独立課金 $40/seat のプロダクトに育ち、44,000+ の学習ルールと 110,000+ リポジトリを背景に解決率 78%。2026-03-25 には Self-hosted Cloud Agents が GA し、2026-03-04 に JetBrains IDE 対応、2026-04-21 には SpaceX とのモデル学習協業が発表された。ARR 20 億ドル、DAU 1M、エンタープライズ売上は 2025 年に 100 倍成長したと公開されている。

**主要機能・プラン**

| 代表機能 | Composer 2、Agents Window、Design Mode、Tab(Fusion、提案回数 21% 減・採用率 28% 増)、Bugbot(独立課金)、Self-hosted Cloud Agents |

**強み**: 対話型の「書く速度」を上げる UI が洗練されていて、VS Code 互換で移行のハードルが低い点は継続的にレビューで称賛されている。日本企業の導入事例(カカクコム 500 人規模、YOUTRUST、Ubie、ココナラ、クラウドビルダーズ)も目立ち、エンタープライズ導入の層が急速に厚くなっている。

**弱み**: 使用量課金の透明性に対する不満は根強く、「料金が割高に感じる」「トークン消費の見え方が分かりにくい」「想定外に課金される」といった声が G2・Product Hunt・Zenn で継続している。2025-06 の料金改定炎上の余波もあって、プラン変更時の告知品質が契約継続率の鍵になりつつある。

### GitHub Copilot ─ GitHub プラットフォームに食い込んだ標準解

Copilot は GitHub エコシステム全体を巻き込む「標準ツール」の位置を固めつつある。2026-04-13 に Data Residency(US + EU)と FedRAMP 準拠が同日発表され、同じ日に Cloud agent の merge conflict を 3 クリックで解決する機能と、CLI セッションを Web/Mobile からリモート制御する preview が公開された。2026-03-05 には Copilot Code Review のエージェンティックアーキテクチャが GA し、2026-02 には GPT-5.3-Codex の LTS(Business / Enterprise 向けに 2027-02-04 まで保証)が開始されている。VS Code Autopilot の public preview、Agent Mode の JetBrains 対応、Custom Agents(`.agent.md`)、Copilot SDK public preview、Copilot Extensions、第三者エージェント委任、GitHub Spark(Vite / React 19+ / TypeScript / Tailwind v4+ / shadcn/ui)といった拡張機能も一気に揃った。一方で 2026-04-20 には Pro / Pro+ / Student の新規サインアップが一時停止され、2026-03 末の PR tips 事件(1.5M の PR に広告的な助言が挿入された件)では GitHub 公式が「判断を誤った」と認めて撤回する局面もあった。

**強み**: IDE 網羅度が飛び抜けていて、VS Code・Visual Studio・JetBrains・Vim・Neovim・Xcode・Eclipse・Zed・Azure Data Studio・SSMS まで対応している。G2 4.5/5.0(227 件、5 星 74% / 4 星 22%)や Gartner 肯定評価に支えられ、Pro $10 は Cursor Pro $20 の半額で「最もコスパが良い」との声が安定的に出ている。Issue・PR・Actions・Runner・Mobile・Firewall といった GitHub プラットフォーム全体の統合は、どの競合も簡単には再現できない。

**弱み**: 個人プランの新規サインアップ一時停止や Opus 4.6 Fast の退役といった経営判断が立て続けに降ってきたことで、短期的な不安が残る。Web agent の 90 秒スピンアップや、2026-02 の RoguePilot 脆弱性(Codespaces で GITHUB_TOKEN 漏洩)、2026-03 末の PR tips 事件といった信頼性・セキュリティ面の事象は、ユーザーが記憶している。

### もう一歩踏み込んだ判断ガイド

**規制業界(金融・医療・政府)で腰を据えて導入するなら**

GitHub Copilot Enterprise と Claude Enterprise の組み合わせが順当になる。FedRAMP 準拠と Data Residency(US + EU)、HIPAA-ready、IP インデムニティ、GPT-5.3-Codex の LTS 保証(2027-02-04 まで)を揃えつつ、Bedrock / Vertex 経由の Claude で IDE 外の長時間タスクに備えられる。導入承認の稟議を通すうえで、二社とも押さえておけば監査・調達側の質問に答えやすい。

**中堅テック企業で GitHub 中心の開発フローを維持したいなら**

GitHub Copilot Business($19/seat)を主軸に置き、複雑タスクや自律的なリファクタは Claude Team Standard($25/seat、Claude Code 込み)に委任する形が現実的だ。Issue・PR 起点のワークフローはそのまま維持しつつ、長時間走らせたい仕事だけ Claude Code に寄せられる。

**スタートアップや個人開発で「書く速度」を最大化したいなら**

Cursor Pro($20)と Claude Pro($17〜20)の二本立てが機能する。Cursor の Tab / Composer で対話的な書く体験を持ち上げつつ、Claude の Skills / Sub-agents で「一晩回しておく」タイプの作業を任せられる。Pro+ $60 や Ultra $200 に上げるのは、実際に自分のトークン消費実績が見えてからで遅くない。

**全社導入でエンタープライズを攻めるなら**
